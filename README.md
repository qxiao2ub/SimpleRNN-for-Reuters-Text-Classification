📘 Sequence Modeling with SimpleRNN – Reuters Classification
📌 Overview

This Jupyter Notebook implements a SimpleRNN-based neural network for text classification using the Reuters dataset. The notebook covers model construction, training, evaluation, and performance analysis within the Keras/TensorFlow framework.

The project demonstrates how recurrent neural networks process sequential text data and highlights their strengths and limitations in multi-class classification tasks.

🧠 Topics Covered

Reuters newswire dataset preprocessing

Text vectorization and sequence padding

One-hot label encoding

SimpleRNN architecture design

Model summary and parameter interpretation

Training & validation performance tracking

Accuracy evaluation on test data

Discussion of RNN limitations

🛠️ Technologies Used

Python

TensorFlow / Keras

NumPy

Matplotlib

Jupyter Notebook

🎯 Learning Objectives

After completing this notebook, readers should be able to:

Build a SimpleRNN model for text classification

Understand input shapes for sequence models

Interpret model summaries and parameter counts

Train RNNs with validation splits

Evaluate classification accuracy on unseen data

Analyze strengths and weaknesses of vanilla RNNs
